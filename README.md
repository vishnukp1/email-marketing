# React + Vite Starter

## Getting Started

### Frontend
- Clone the repo.
- Run `npm install` and `npm run dev`.

### Backend
- Run `npm install` and `npm start`.

## Packages Used

### Frontend:
- Tailwind
- Material-UI
- React Flow
- Axios
- uuid

### Backend:
- Express
- Mongoose
- jsonwebtoken
- cors
- nodemon
- bcrypt
